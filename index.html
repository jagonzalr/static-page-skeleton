static-page-skeleton

A skeleton for a static web page with Gulp and Skeleton.

Styles

  • Normalize CSS
  • Skeleton Responsive CSS Framework

Batteries

  • Browsersync
  • Inject of CSS and JS
  • Minification

Usage

  • Download the zip file
  • Install dependencies npm install
  • Run project npm start or gulp
  • To build for distribution / production gulp dist
